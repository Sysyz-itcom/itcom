# 希沃课程表

为了更方便同学和老师关注课堂信息，我们使用ClassIsLand来展示课程表，目前仅**邵阳市第一中学高一年级和高二年级适用**，如果您有需要，您可以 [点击此处](https://share.feijipan.com/s/A6C6Irtt) 进行下载。

## 注意事项

::: details 如何使用
下载压缩包并解压，将**整个文件夹**复制到一体机的 ` D 盘` 中，然后打开文件夹，运行 `ClassIsland.exe` 即可。
:::

::: details 为什么处于集控状态
通过集控可以快速为所有高一和高二班级的时间表进行控制，避免带来二次问题，请您放心，集控不会收集任何电脑信息。
:::

::: details 基础教程
信息委员（电教委）可以通过官方发布的教程视频进行更改，集控平台为各班级提供了一份可以快速修改的模板。[教程传送门](https://www.bilibili.com/video/BV1fA4m1A7uZ)
:::
